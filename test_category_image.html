Technical Details

CSS Fix Applied:

.bg-light-subtle .img-fluid.float-end {
  max-width: 140px;
  max-height: 140px;
  width: auto;
  height: auto;
  object-fit: cover;
}

What this does:

  • max-width/max-height: 140px - Constrains the image to never exceed the intended display size
  • width/height: auto - Maintains aspect ratio
  • object-fit: cover - Ensures the image fills the space nicely without distortion
